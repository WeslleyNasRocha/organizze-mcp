# Security Policy

## Supported Versions

Only the latest version of Organizze MCP is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead, please send an email to the maintainer or use the GitHub Private Vulnerability Reporting feature if available.

We strongly advise you to never share your `ORGANIZZE_TOKEN` or `ORGANIZZE_EMAIL` in any public forum, issue description, or logs.
